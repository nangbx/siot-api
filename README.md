# SIOT project

## Requirement

- `Typescript`
- `Mongodb`
- `Mongoose`
- `ExpressJs`
- `NodeJs`

## Quick start

1. Make sure you already installed `Nodejs` in system.
2. Run command `npm i` for install all dependances.
3. Create a `.env` file
4. Copy and edit your `env` from `.env.example` to `.env`
5. Run command `npm start` for start development server
